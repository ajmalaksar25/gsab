# GSAB realtime demo — v5 storyboard (contradiction-first, shareable)

Built on the framework (`VIDEO-FRAMEWORK.md`). **Viral thesis:** *"This app has no
database — the backend is a Google Sheet."* (shareable as **"No DB. Still live."**). The
feeling we're after: *the kind of wrong that's actually useful.* ~25s. Hook in 3s; a new
payoff every ~3s. Demo stays **real** (real Sheet, real app, real ~1s polling).
